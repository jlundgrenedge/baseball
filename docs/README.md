# Documentation

Development documentation for the Baseball Physics Simulator.

## Structure

### `/guides/`
How-to guides and tutorials for working with the simulation:
- **Database integration** - PyBaseball, CSV export
- **Tuning guides** - K%, BB%, simulation metrics
- **Performance** - Optimization, CPU usage
- **Logging** - Debug output reference

### `/reference/`
Technical reference documentation:
- **Architecture** - System design, component interactions
- **Ballpark/Field** - Hit classification, coordinate system
- **Statcast** - Metrics analysis, calibration data

### `/implementation/`
Feature implementation notes (current work):
- **Physics calibration** - Spin, Reynolds number
- **Attributes** - Player skill mappings
- **Game systems** - Baserunning, pitching enhancements

### `/archive/`
Historical documentation from completed development phases:
- Sprint/Phase implementation logs
- One-off diagnostic analyses
- Dated fix reports

## Finding Documentation

| Looking for... | Go to... |
|----------------|----------|
| How to do something | `/guides/` |
| How something works | `/reference/` |
| Current feature work | `/implementation/` |
| Historical context | `/archive/` |

## Key Files

- **CLAUDE.md** (root) - Comprehensive AI assistant guide
- **TESTING_STRATEGY.md** (root) - Testing workflow and methodology
- **Full Architecture Map.md** - Complete system architecture
